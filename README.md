## How to start (local dev)
1) In one terminal:
```
cd apps/api
cp .env.example .env
# Fill env vars
pnpm i
pnpm dev
```
2) In another terminal:
```
cd apps/web
cp .env.example .env
# Fill VITE_API_BASE_URL
pnpm i
pnpm dev
```
